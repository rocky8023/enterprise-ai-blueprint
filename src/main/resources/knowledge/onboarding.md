# 入职流程

## 入职前（HR 准备）

- 入职前 5 个工作日：HR 发送 Welcome Email，附入职须知、所需材料清单
- 入职前 3 个工作日：HR 通知 IT 准备设备、行政准备工位、直属主管准备入职计划
- 入职前 1 个工作日：HR 确认到岗时间，发送报到地址与联系人

## 入职当天

| 时间 | 事项 | 负责人 |
|---|---|---|
| 9:00 | 报到、签订劳动合同 | HR |
| 10:00 | 公司介绍 + 制度宣讲 | HR |
| 11:00 | IT 配置设备、开通账号 | IT |
| 14:00 | 部门内部介绍 | 直属主管 |
| 15:00 | 工位安顿、文化产品（公司 T 恤 + 笔记本）发放 | 行政 |
| 16:00 | 第一天工作启动会 | 直属主管 |

## 入职 30/60/90 天

- **30 天**：完成新人培训（包含合规、信息安全、产品概览）+ 第一份产出（项目调研报告或 demo）
- **60 天**：独立承担一个小模块的交付
- **90 天**：转正答辩，由直属主管 + HR + 跨部门 1 位评委评定

## 试用期权益

试用期工资为正式工资的 80%-100%（依岗位定），转正后从入职次月起补足差额。
